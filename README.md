### ✨ Features

#### 🧠 AI-Powered Insights
- **Chillzz AI Stylist**: Personal fashion consultations using Google Gemini 2.5 Flash for high-fidelity recommendations.
- **Dynamic Search**: Context-aware product discovery refined through advanced backend filters.

#### 🛡️ Secure & Seamless Experience
- **Elite Demo Mode**: A premium, KYC-free transactional gateway simulating bank-level security for portfolio sharing.
- **Hybrid Authentication**: Secure Login/Signup with Google OAuth 2.0 and custom JWT workflow.
- **Cloud Infrastructure**: Scalable backend on Render and lightning-fast frontend delivery via Vercel.

#### 💎 Boutique Aesthetic
- **Cinematic Branding**: High-fidelity video hero section with smooth overlays and premium typography (*Playfair Display*, *Jost*).
- **Responsive Navigation**: Robust SPA routing optimized for modern browser performance.
- **Rich Interaction**: Gentle Framer Motion transitions and hover micro-animations.

### 🛠️ Installation & Setup

#### Prerequisites
- Node.js (v18+)
- MongoDB Atlas Account
- Google AI (Gemini) API Key
- Google Cloud OAuth Credentials

#### Backend Setup
1. `cd backend`
2. `npm install`
3. Create a `.env` file:
   ```env
   PORT=5000
   MONGO_URI=your_mongodb_uri
   JWT_SECRET=your_secret
   GEMINI_API_KEY=your_key
   EMAIL_USER=your_email
   EMAIL_PASS=your_pass
   ```
4. `npm start`

#### Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`
