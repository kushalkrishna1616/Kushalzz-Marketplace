# Kushalzz Marketplace

Welcome to the **Kushalzz Marketplace** source code! This is the fresh, newly designed frontend repository.

## Starting the Project Locally

To run the beautifully redesigned frontend, you can use the startup script:

```bash
.\run_km_marketplace.bat
```

Alternatively, you can manually start the frontend server:

```bash
cd frontend
npm run dev
```

## Features
- Fully responsive modern UI
- Custom apparel storefront
- Premium "Kushalzz Marketplace" branding
